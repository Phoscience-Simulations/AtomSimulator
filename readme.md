# Audio Visualiser
This Current Project is Finished.
This is an Audio Visualiser where Particles move and effects whenever there is a beat in the music. 
This was a fun project, and it ended well :).

## Controls
E:     Increase Blur\
Q:     Decrease Blur\
A:     Back Song\
S:     Shuffle Toggle\
D:     Next Song\
SPACE: Pause VIsualiser\
P:     Pause Music
